# 多媒体技术大作业 - 网页前端设计说明

## 1. 项目概述
本项目旨在设计并实现一个“智能机械臂垃圾分类”实验微课平台。平台集成了教学视频、图文步骤详解以及核心算法的交互式演示，帮助学生直观理解基于大模型（LLM）与机器视觉（VLM）的机械臂控制原理。

## 2. 栏目设计与功能
网页主要分为以下四个模块：

### 2.1 首页与微课视频 (Micro-lesson Video)
- **功能**：展示课程标题及教学视频播放器。
- **设计**：使用 Bootstrap 的 Card 组件封装视频区域，提供简洁的播放界面。
- **内容**：预留视频位，用于讲解机械臂组装与环境配置全流程。

### 2.2 实验步骤详解 (Step-by-Step Guide)
- **功能**：通过选项卡（Tabs）形式展示实验的四个关键阶段：
    1. 硬件组装
    2. 环境部署
    3. 核心代码开发（ReAct Agent）
    4. 手眼标定原理
- **设计**：左侧为步骤导航，右侧为详细内容区。代码块采用高亮背景，便于阅读。

### 2.3 算法交互演示 (Interactive Demo)
- **核心创新点**：利用 HTML5 Canvas 技术开发了一个 2D 仿真工作台。
- **交互逻辑**：
    1. 用户在输入框输入自然语言指令（如“把蓝色方块扔进垃圾桶”）。
    2. 系统模拟 Agent 的思考过程，在右侧“系统日志”中实时打印 ReAct 循环（Thought -> Action -> Observation）。
    3. 视觉反馈：Canvas 上的机械臂自动移动、抓取物体并投放至指定区域，直观展示坐标变换与动作执行过程。
- **技术实现**：
    - 使用 JavaScript `requestAnimationFrame` 实现流畅的机械臂运动动画。
    - 模拟 VLM 视觉识别过程，计算物体坐标。

### 2.4 实验报告 (Report)
- **功能**：提供实验总结与心得体会的展示区域。

## 3. 技术栈
- **前端框架**：Bootstrap 5.3 (响应式布局，美观且兼容性好)
- **核心语言**：HTML5, CSS3, JavaScript (原生 ES6+)
- **图标库**：FontAwesome 6.0 (增强视觉体验)

## 4. 使用说明
1. 进入 `web_project` 文件夹。
2. 双击打开 `index.html` 文件（建议使用 Chrome 或 Edge 浏览器）。
3. 在“交互演示”区域，尝试输入指令或点击画布添加新物体进行测试。

## 5. 对应作业要求
- **多媒体工具应用**：网页集成了视频、文本、图形动画等多媒体元素。
- **交互页面设计**：实现了算法的动态演示与用户输入响应。
- **内容深度**：涵盖了手眼标定、大模型调用等技术细节，参考了实验三报告与开发实例。
